## Agents

- **decomp-evaluator**: Evaluate whether a dependency should be kept or decomposed. Use during dependency decomposition (phase 1) to assess each library in the queue.
- **decomp-implementer**: Implement or replace a dependency in diy_<package>/ based on a decomposition evaluation. Use during dependency decomposition (phase 1) after evaluation.
- **test-discoverer**: Search reference test suite for tests relevant to a target function. Use during test curation (phase 0) to discover existing tests from the original library.
- **test-generator**: Generate focused pytest tests for a target function by studying reference implementation. Use during test curation (phase 0) after test discovery.
